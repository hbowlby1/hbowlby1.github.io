Object Methods

  • A function that is a property of an object is a method.
  • You can use method shorthands for example: 
    // these objects do the same

                user = {
                  sayHi: function() {
                    alert("Hello");
                  }
                };
                
                // method shorthand looks better, right?
                user = {
                  sayHi() { // same as "sayHi: function(){...}"
                    alert("Hello");
                  }
                };
  • "This" is a keyword that can be used to call the object. For example, this.name is accessing the user object and the name.
  • "This" is not bound and can be called outside of the object.
  • Arrow functions cannot use "This"

This in JavaScript

I already understand this but read "this" for more clarification in JavaScript.
  • you can call "This" in 6 different ways.
    1. in global context
      • you don't really use this version of "this". It calls the Window object.
    2. in object construction
      • this is used with constructors.
    3. in an object method
      • used to get the instances of an object
    4. in a simple function
      • if you call this in a simple function it refers to the Window object.
      • to fix this use a variable to call "this" (usually you use self. Or use an arrow function.
    5. in an arrow function
      • using a this in an arrow function calls the object instead of the Windows object.
    6. in an event listener
      • this refers to the event listener item
      • you still need to provide a reference (variable)
      • to call the object instead of the event you need to use bind. For example this.listener = this.speakLeet.bind(this)
        • bind is similiar to putting parameters in a function
        • it does not work with arrow functions

  • use the ES6 since it is easier const ironMan = {name, realname};

  • make sure to add parenthesis after calling object methods. ( super.fly() )
  • You can loop through an objects methods by calling a for in loop for(const key in superman) { do something }
  • you can add properties to an object by using this format object.newProperty = propertyItem
  • objects are not ordered lists
  • you can update properties like this super['real name'] = 'Kal-El'

  • you can nest objects and call them like jla.batman.realName

Chapter 6 JavaScript Novice to Ninja

  • While JavaScript is the main language for DOM manipulation, other languages can do it too.
  • Node lists are array-like objects but are not arrays.
    • use .from() to turn it into an array for array methods.
  • document.getElementById() is the most common way of grabbing items in a document
  • document.getElementByTagName() gets objects based on the tag.
  • document.getElementByClassName() as it says, gets an object by the class name.
  • document.querySelector()lets you use CSS notation for grabbing tags, id's or class. It's personally my favorite method.
  • you can use .textContentto grab the text info in a tag
    • it does not work in IE 8 so you can use innerText
  • you can use .toggle() to remove classes. It's useful for areas like dropdowns and color changing from my experience.
  • every element has a style property so you can change items dynamically using CSS. example: superman.style.border = "red 2px solid"; just know it adds it via inline CSS.
  • if you use the style method any CSS properties seperated by a dash must instead be camel case. Example: background-color = backgroundColor
  • you can also use bracket notation superman.style['background color']
