JavaScript: Novice to Ninja, 2nd Edition

Chapter 2 Notes

  • var was replaced with let due to scoping issues. It's better to use let and const.
  • Variables with an underscore at the beginning are usually private variables.
  • putting $ at the beginning is usually associated with jquery
  • startsWith() is a method you can use to check if the string starts with a certain character. It is case sensitive.
  • You can use typeOf() is determine the primitive value. It cannot distinguish between integers and floats. You can use isInteger() for that.
  • toFixed(x) will round up the last number indicated by the parathenses.
  • null behaves like zero but undefined is NaN
  • Always use hard equality (===)

Chapter 3 Notes

  • join() is used to turn items in an array into a string. Whatever is in the parathenses is what seperates them.
  • You can use reverse() on arrays to reverse the items in it. It changes the order permanently.
  • sets were introduced in ES6 and acts like an array but any number that repeates only appears once.
  • rest of the chapter after this is review.

Chapter 4 Notes

  • arrow functions are always anonymous so you have to assign them to a variable.
  • arrow functions are very versatile and sometimes better because they are easier to read, require only one line and the return statement is applied.
  • function hoisting is where you place the function before it has been defined.
