Question

Since mobile is the future, should developing a mobile site come before developing a desktop site?

Notes

  • you can store information using localStorage command
  • localStorage allows information to be saved even if the user exits the page.
  • More on localStorage can be found
  • Start building a website from a mobile view.
  • The hamburger menu is not known by everyone, so including both the button and the word "menu" helps.
  • Autocomplete is important when it comes to forms because of how easy it is to fail on typing information in
  • Being able to see a password is important due to the common issue of entering passwords wrong.
  • Don't just use standard keyboards, for instance, phone numbers.
  • ads need to be simple and smaller.
  • not mentioned but lazy loading is important for ads
  • never use pop-ups
  • content comes first
