Chapter 9: The Window Object

  • Every browser window, tab, popup, frame, and iframe has a window object.
  • In general, you should refer to global variables without using the window object; itâ€™s less typing and your code will be more portable between environments.

  • Today you usually wouldn't use a lot of the windows methods due to annoyance, pop up blockers and web browsers preventing certain action. In other words, use caution.
  • window.screen can be useful on phones.
  • cookies are a way of storing information that pertains to the user. You can view current cookies either by using the web developer extension or typing document.cookie in the console.
  • you can set the expiration on a cookie, if you don't they will be removed when the session is closed or the web browser is closed.
