Chapter 8: Transforms and Transitions

  • To get transform to work in older browsers make sure to include:
  • -webkit-transform: translate(45px,-45px); /* iOS8, Android 
            â†µ4.4.3, BB10 */
            -ms-transform: translate(45px,-45px); /* IE9 only */
            transform: translate(45px,-45px);
  • translate does not work with inline elements.
  • Translate is for moving items along the x and y axis
  • transforming is for scaling content to make it appear bigger or smaller.
  • rotate is used to rotate items
  • each of these can be put on the same line:
    • I.E: transform: rotate(10deg) translateX(40px) scale(1.5)
  • skew, as it says, allows us to to move the object on the x and y axis. I like to think of this as y=mx+b.
  • you can change the origin (or center point) of an object by using transform-origin
  • Animation/transitions, can be smoother and sometimes better then JavaScript animations.
  • steps to create a simple transition in CSS
    1. Declare the original state of the element in the default style declaration.
    2. Declare the final state of your transitioned element; for example, a :hover state.
    3. Include the transition functions in your default style declaration using the transition properties, including: transition-property, transition-duration, transition-timing-function, and transition-delay.
  • a good example of transform, animations and more in practice is on my CSS Zen Garden web page.

Chapter 12: Canvas, SVG, and Drag and Drop

  • We can use JavaScript to make pictures by using Canvas tags. Just make sure to include width, height and either an ID or class or both.
  • There is mainly only support for 2D when using the Canvas api. For 3d you can use webGl
  •             1: var canvas = document.getElementById("myCanvas"); 
            2: var context = canvas.getContext("2d"); 
            3: context.strokeStyle = "red";
            4: context.fillStyle = "rgba(0, 0, 255, 0.5)";
            5: context.fillRect(10, 10, 100, 100);   
            6: context.strokeRect(10, 10, 100, 100);
    • above makes a rectangle. Line 1 is grabbing the canvas item.
    • Line 2 establishes the type of drawing.
    • Line 3 is the color of the stroke.
    • line 4 is the color that the rectangle is filled with
    • line 5 is where the rectangle is created and how big it is, the parentheses is (x, y, width, height)
    • line 6 is another rectangle of the same size and place but it is not filled. Hence, the way we get the red stroke.
  • You can use arc(x, y, startAngle, endAngle, anticlockwise) for a circle. EX: context.arc(50, 50, 30, 0, Math.PI*2, true);
  • you can save images created by canvas. You can use window.open(canvas5.toDataURL("image/png"));
