  • you can use anonymous functions to define and even rewrite functions.
  •             function party(){
                console.log('Wow this is amazing!');
                party = function(){
                    console.log('Been there, got the T-Shirt');
                }
            }
            
            const beachParty = party; // note that the party function has not been invoked
            
            beachParty(); // the party() function has now been redefined, even though it hasn't been called explicitly
            << 'Wow this is amazing!'
            
            party(); 
            << 'Been there, got the T-Shirt'
            
            beachParty(); // but this function hasn't been redefined
            << 'Wow this is amazing!'
            
            beachParty(); // no matter how many times this is called it will remain the same
            << 'Wow this is amazing!'

Chapter 13: Ajax

  • Ajax allows web pages to communicate asynchronously with a server. It can create new webpages wihtout reloading.
  • We can use the fetch method to grab an API
  • const url = 'https:example.com/data';

fetch(url)
.then((response) => {
    if(response.ok) {
        return response;
    }
    throw Error(response.statusText);
})
.then( response => // do something with response )
.catch( error => console.log('There was an error!') )
  • above is a way to call the fetch method in order to retrieve an API.
